2. What is the major difference between Git and GitHub?

GIT is a version control system where users can keep track of their code and history. GitHub is a website in which you can upload those gits and you (and others) can view the changes in your code as well as make changes and collaborate.

3.  What are the steps to fork a GitHub repository?
To create a fork you first go to the repo you would like to duplicate. You then will click "fork" in the top right corner. You then will name your fork or "branch" as well as put a description. You then will be able to copy the main branch and create your fork.
